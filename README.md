# Hi there, I'm berrywat! 👋

🍓 **Data-driven Strawberry Smart Farmer based in Jeju, Korea**

저는 제주의 아름다운 자연 속에서 첨단 스마트팜 기술을 활용해 최고 품질의 딸기를 재배하고 있습니다. 
농업 현장에서 발생하는 다양한 IoT 데이터, 환경 제어 시스템, 그리고 육묘/재배 데이터를 분석하고 스마트 농업 기술을 연구합니다.

## 🛠️ Tech & Interests

- **Data & Tech:** Agriculture Data Analysis, Smart Farming Integration
- **Main Crop:** Strawberry (육묘, 재배, 시설 관리)
